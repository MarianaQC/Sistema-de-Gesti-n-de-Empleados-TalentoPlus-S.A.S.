# Employee Management System - TalentoPlus S.A.S.

## Project Description

This is a comprehensive Human Resources and Catalog Management System developed in **ASP.NET Core 8.0** using a Clean Architecture approach. The project features full staff administration (CRUD) and the crucial **Mass Employee Import** functionality from an Excel file.

Key functionalities include dynamic column mapping, transactional catalog management (Departments, Positions), and advanced features like an AI-powered dashboard and secure employee authentication.

This system was developed by **Mariana Quintero Cardona** (Clan: **Hopper**).
